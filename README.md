# NEON DELVE — 電脳塔の探索者

『不思議のダンジョン 風来のシレン6 とぐろ島探検録』のゲームルールを調査し、その核となるシステムを**近未来サイバーパンク世界**に置き換えたブラウザ製ローグライクRPGです。

> 西暦2099年。暴走したAI《セルペンス》が支配する電脳塔。
> その最深部・地下20階に眠る「シンギュラリティコア」を求め、流浪のランナーは塔へ潜る——

**📱 スマホ対応**: タッチ操作（画面下の方向パッド+アクションボタン）と、プレイヤー追従カメラによる拡大表示に対応しています。PCでは従来どおりフロア全体を表示します。

## 遊び方

ビルド不要です。`index.html` をブラウザで開くだけで遊べます。

```bash
# ローカルで開く場合
open index.html        # macOS
xdg-open index.html    # Linux

# またはローカルサーバーで
python3 -m http.server 8000
# → http://localhost:8000
```

### 操作方法

**スマホ/タブレット**: 画面下の方向パッド（8方向+中央=足踏み）で移動・攻撃、「道具」「拾う」「階段▼」「決定」ボタンで操作します。メニューは項目をタップで選択できます。

**キーボード**:

| キー | 操作 |
|---|---|
| 矢印キー / W A D X | 移動（敵の方向へ移動で攻撃） |
| Q E Z C | 斜め移動 |
| S または . | 足踏み（ターン経過・HP回復） |
| I | 持ち物を開く（使う/装備/投げる/置く） |
| G | 足元のアイテムを拾う |
| > | 降下シャフト（階段）で次の階へ |
| Esc | メニューを閉じる |

## シレン6から再現したルール

ネット調査（シレン6攻略wiki・ファミ通・公式サイト等）で確認した仕様をベースにしています。

| シレン6の要素 | 本作での対応 |
|---|---|
| ターン制・ランダム生成ダンジョン | 1入力=1ターン。フロアは毎回自動生成（部屋+通路+階段） |
| 倒れると Lv1・アイテム全ロスト | ゲームオーバーで持ち物・クレジット・レベルを失い再挑戦 |
| 満腹度（最大100、10ターンで1減、0でHP減少） | **充電率**。仕様値はシレン6と同じ |
| 満腹時に食べると最大値上昇（+3/+4、上限200） | エナジーパックで最大充電率が上がる |
| **ドスコイ状態**（満腹150以上で攻撃1.5倍・罠無効） | **オーバーチャージ状態**。発動150%・解除120%も同値 |
| ダメージ計算式 `攻撃力×乱数 − 防御/2 + 1` | 同式を採用（攻撃力 = ちから + Lv補正 + 武器×(0.75+ちから/32)） |
| 初期ステータス Lv1 / HP15 / ちから8 | 同値 |
| レベルアップで最大HP+3〜6（ランダム） | 同値 |
| 自然回復はレベル帯で増加・攻撃ターンは回復しない | 同仕様（Lv1-9 / 10-19 / 20+ で増加） |
| 未識別アイテム（周回ごとにランダムな見た目名） | 薬剤・チップ・ガジェットは未識別。使用かスキャンで判明 |
| 草・巻物・杖（回数制）・おにぎり・ギタン | ナノ薬剤・データチップ・ガジェット［回数］・エナジーパック・クレジット |
| 草を飲むと満腹度+2 | 薬剤で充電率+2 |
| 杖は直線上の敵に効果（かなしばり/場所がえ/吹き飛ばし/鈍足） | スタナー（攻撃まで持続）/テレポーター/ブラスター/ジャマー |
| 復活の草 | バックアップチップ（所持していると一度だけ完全復活） |
| **共鳴**（特定の武器+盾セットでボーナス） | パルスブレード+パルスバリアで「パルス共鳴」攻+3 防+3 |
| **デッ怪**（規格外の巨大モンスター） | ビヒモス・ギガメック（低確率で出現する超強敵・撃破で大量経験値） |
| モンスターハウス | ホスタイルネスト。一定確率で発生し、入室すると敵が一斉に起動する |
| モンスターが他者を倒して成長する系統 | デバウラー（隣接した機体を取り込んで強化） |
| 罠（地雷・落とし穴・ワープ・睡眠ガス・サビ等、踏むまで不可視） | 8種の罠を再現（オーバーチャージ中は踏み潰して無効化） |
| 突風（1700/1800/1900警告 → 2000ターンで強制失敗） | 同値（とぐろ島と同じタイマー） |
| 斜め移動は角を抜けられない | 同仕様 |
| 倍速・鈍足・混乱・睡眠・かなしばり等の状態異常 | 実装済み（かなしばりは攻撃されるまで持続） |
| **店と泥棒**（未払いで店を出ると店主が豹変・階段に盗賊番） | マーチャントドロイドの店。購入/売却/返品、未払いで店を出ると戦闘形態+執行ユニットが階段を封鎖。逃げ切れば強奪成功。店内に罠は無い（シレン6準拠） |
| **壺**（保存の壺・合成の壺、容量制、割ると中身が出る） | ストレージコンテナ（出し入れ自由）と合成コンテナ（同じ装備の強化値+回数を合成・筐体を破壊して取り出す）。投げても壊れる |
| とぐろ島の最深部を目指す | 地下20階の「シンギュラリティコア」奪取が目標。守護ボス（セルペンス・アバター）付き |

### 本作独自のアレンジ

- 舞台・アイテム名はすべて近未来テーマ（おにぎり→エナジーパック、ギタン→クレジット等）
- ハッキングカード（投擲アイテム）は本作独自の追加要素（シレン6では札は廃止されているため、シレン4/5の札へのオマージュ）
- 印（武器に効果を付与するシール）・仲間・救助システムは未実装（今後の拡張候補）

## 開発

```bash
node test/smoke.js   # ヘッドレスのスモークテスト（43項目）
```

| ファイル | 内容 |
|---|---|
| `js/rng.js` | シード対応乱数（mulberry32） |
| `js/dungeon.js` | フロア生成（3x3グリッド分割+通路接続） |
| `js/data.js` | モンスター/アイテム/罠/経験値テーブル定義 |
| `js/game.js` | コアロジック（ターン制・戦闘・AI・状態異常・突風） |
| `js/assets.js` | 画像アセットローダー（無ければ記号描画にフォールバック。仕様は `assets/README.md`） |
| `js/render.js` | Canvas描画・追従カメラ・シレン式視界（部屋単位の可視判定） |
| `js/ui.js` | ステータス/ログ/インベントリメニュー |
| `js/main.js` | 入力処理・起動 |
