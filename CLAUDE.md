# CLAUDE.md — NEON DELVE 開発ガイド

## プロジェクト概要

『風来のシレン6』のルールを近未来サイバーパンク世界に置き換えたブラウザ製ローグライクRPG。
ビルド工程なし（バニラJS）。`index.html` を開くだけで動く。GitHub Pages で公開中:
https://hikavn.github.io/shiren/

## コマンド

```bash
node test/smoke.js   # ヘッドレスのスモークテスト（必ず通すこと）
node --check js/*.js # 構文チェック
```

## 構成

| ファイル | 役割 |
|---|---|
| `js/rng.js` | シード対応乱数 |
| `js/dungeon.js` | フロア生成（グリッド分割+通路） |
| `js/data.js` | モンスター/アイテム/罠/経験値テーブル定義 |
| `js/game.js` | コアロジック（ターン制・戦闘・AI・店・コンテナ・突風） |
| `js/render.js` | Canvas描画・追従カメラ・シレン式視界 |
| `js/assets.js` | 画像アセットローダー（無ければグリフ描画にフォールバック） |
| `js/ui.js` | ステータス/ログ/メニュー（キーボード+タップ両対応） |
| `js/main.js` | 入力処理・起動・rAFループ |
| `assets/` | 画像アセット置き場（仕様は assets/README.md） |
| `test/smoke.js` | Node実行のロジックテスト |

## 開発方針・規約

- **ビルド不要を維持する**（npm依存・バンドラ禁止。外部ライブラリはCDNの場合のみ可だが現状ゼロ）
- ゲーム内テキストは日本語。**和物系の語（壺・泥棒・神髄など元ネタを想起させる語）は使わず**、
  サイバーパンク世界観の語に置き換える（例: 壺→コンテナ、神髄コア→シンギュラリティコア）
- シレン6由来のルール数値（満腹度・突風タイマー・ダメージ式など）は変更しない。
  対応関係はコードコメントと README の対照表に記録
- ロジック変更時は `test/smoke.js` にテストを足す。描画はNodeで検証できないため、
  ユーザーの実機スクリーンショットで確認する
- グラフィック路線: **リッチ重視の2D（HD-2D風）**。疑似3D壁・動的ライティング・
  移動補間・パーティクル・ダメージポップ・画面シェイクを Canvas 2D で実装済み。
  高品質ドット絵アセットを assets/ に差し込んで完成度を上げる。
  さらなる表現（法線マップ風ライティング等）が必要になったら PixiJS(WebGL) 移行を検討
- 描画エフェクトは `game.addFx({type, ...})` でキューに積み、render.js が消費する
  （type: pop / burst / ring / beam / shake）
- ブランチ `claude/shiren-roguelike-game-s7od11` で開発し、PRを作成。
  マージはユーザーの指示（「マージして」）で行う。マージすると GitHub Pages へ自動デプロイ

## アセットパイプライン

1. ユーザーが `assets/README.md` の仕様どおりに PNG を用意し、リポジトリにコミット
2. `assets/manifest.json` に登録（ファイル名・フレーム数・fps）
3. `js/assets.js` が起動時に読み込み。登録が無い/読めないものは従来のグリフ描画に自動フォールバック
